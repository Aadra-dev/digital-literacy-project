# digital-literacy-project
## Project Overview

This repository contains my Digital Literacy Project completed as part of Semester 2 coursework. The project is divided into five tasks, each mapped to a specific module of the course.

The goal of this project is to develop essential digital skills including:
-Creating visual content and presentations
-Building a professional digital presence
-Practicing coding and using collaboration tools
-Professional communication and online etiquette
-Cybercrime awareness and safe digital practices


Each task demonstrates practical application of digital tools and concepts that are important for academic and professional growth.

## Module-wise Summary
### Module 1: Presentation & Visual Design Tools

This module focused on hands-on experience with modern presentation and design tools.

Skills & Tools Used:

-Canva, Prezi, Miro, CoolText, GIF creation tools
-Designing infographics and visual content
-Communicating ideas effectively through visuals

Task Completed:
Created a Digital Literacy Awareness Infographic explaining key concepts such as digital literacy, online safety, and professional presence.

### Module 2: Building a Digital Portfolio

This module emphasized creating a professional online presence across multiple platforms.

Skills & Platforms Used:

-GitHub
-LinkedIn
-Other platforms (e.g., Kaggle / Stack Overflow / ResearchGate)

Task Completed:
Set up and updated professional profiles
Created a GitHub README profile repository
Structured portfolio for academic and career use

### Module 3: Coding & Collaboration Platforms

This module introduced coding practice platforms and collaborative tools.

Skills & Platforms Used:

-Coding: HackerRank / CodeChef / HackerEarth
-Collaboration: Google Forms (Google Workspace)

Task Completed:
Completed a beginner-level coding challenge

### Module 4: Email & Social Media Etiquette

This module focused on professional communication and responsible online behavior.

Skills Covered:
-Writing formal and professional emails
-Understanding email etiquette
-Social media best practices

Task Completed:
Wrote two professional emails:
Request for assignment deadline extension
Application for a summer internship opportunity
Created a Social Media Do’s and Don’ts List highlighting responsible and professional online conduct

### Module 5: Cybercrime Awareness & Online Safety

This module emphasized awareness of cyber threats and safe digital practices.

Skills Covered:

Understanding cybercrime and financial fraud
Identifying online threats
Preventive measures and digital safety

Task Completed:

Developed a Cybercrime Case Study based on a real UPI fraud incident involving an IIT student
Created a Prevention Checklist to promote safe online financial practices


Created a Digital Literacy Quiz using Google Forms
Learned collaborative and cloud-based workflows
